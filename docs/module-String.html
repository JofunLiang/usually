(inner) mask(str, startopt, endopt, maskopt) → {string}

Source:
  • string.js, line 114
使用指定的掩码字符替换start~end之间的所有字符
Example
U.mask(123456789) // => *********
U.mask(123456789, 3) // => 123******
U.mask(str, 0, 4) // => *****6789
U.mask(str, 3, 4) // => 123**6789
U.mask(str, 3, 4, '&') // => 123&&6789
Parameters:

 0 可选，结束位置，默认为0（即字符串结尾）
mask string <optional>
 '*' 可选，掩码字符，默认为'*'号
